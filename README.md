# Gym Progress Tracker

A mobile-first fitness tracking app — browse exercises, build workout plans, log sessions, track goals, and visualize progress over time. Built with React, Vite, and Firebase.

## Features
- Google Sign-In authentication
- Exercise library with search and filters
- Custom workout plan builder
- Live workout session logging with rest timers
- Goals and body measurement tracking
- Progress charts (volume, body weight)
- Installable PWA, dark mode by default

## Stack
Vite + React, Firebase (Auth + Firestore), React Router, Recharts, Lucide icons.
